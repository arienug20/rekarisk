# Rekarisk — Methodology Reference

> **Status:** This document will contain all equations, model descriptions, and references. To be written during Phase 15 (Documentation).

## Models Covered

### Source Term
- Bernoulli liquid orifice
- Choked/unchoked gas orifice
- Two-phase flow (HEM, slip models)
- Pipe flow (Darcy-Weisbach)
- Vessel blowdown
- Pool spreading & evaporation
- Rainout & aerosol fraction

### Dispersion
- Gaussian plume (continuous)
- Gaussian puff (instantaneous)
- Dense gas (SLAB-based)
- Building wake / cavity
- Lift-off / transition
- Indoor infiltration

### Fire
- Pool fire (Mudan, Shokri-Beyler)
- Jet fire (API RP 521)
- BLEVE / fireball (HSE, CCPS)
- Flash fire (LFL isopleth)
- Line-of-sight blocking

### Explosion
- TNT equivalency (Kingery-Bulmash)
- TNO Multi-Energy
- Baker-Strehlow-Tang
- Obstacle effects

### Vulnerability
- Probit models (thermal, overpressure, toxic)
- Toxic load (Cⁿ·t)
- Shelter factor

### QRA
- Event tree analysis
- Individual risk (IRPA)
- Societal risk (FN curve)
- Risk matrix (ISO 17776)
- ALARP

Full methodology with equations, assumptions, and references will be provided in the complete document.
