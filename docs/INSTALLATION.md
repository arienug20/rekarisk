> **Status:** This document will be written during Phase 15 (Documentation).

## Quick Install (from source, development)

```bash
git clone https://github.com/arienug20/rekarisk.git
cd rekarisk
pip install -r requirements.txt
pip install -e ".[dev]"
pytest
```

## System Requirements

- Python 3.11+
- 4 GB RAM (8 GB recommended)
- Windows 10+, Ubuntu 22.04+ / Debian 12+, or macOS 12+
